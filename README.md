# Ook!
___

### Prerequisites
You need a C compiler, CMake and Git.
```shell
apt install build-essential cmake git
```
___
### How to use
Clone the repository and build it:
```shell
git clone https://github.com/iKryxx/eso-ook.git
cd eso-ook
mkdir build && cd build
cmake ..
make
```
